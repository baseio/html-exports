HTML Exports Custom Element Demo

HTML exports gives us a ton of flexibility around how modules are organized. This demo serves as an example of organizational methods, by focusing on a custom element.

Each example exports a prototype for the element, which we register directly via this document.

<squid-es5-combined>

Both the element prototype and its templates are declared in the same file (squid-es5-combined.html), with an ES5 element definition.

<squid-es5-split>

Another option is to structure your code as an ES5 module that imports its templates separately. See squid-es5-split.js for a CommonJS approach.

<squid-es6-split>

A third option is to declare your element as an ES6 module and class that imports its templates separately. See squid-es6-split.js for this approach.
